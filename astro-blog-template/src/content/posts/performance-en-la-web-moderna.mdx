## La percepción importa

Una web lenta se siente rota, aunque funcione.

La performance impacta en:
- UX
- SEO
- conversión
- credibilidad

---

## Menos JavaScript

Más JS no significa mejor experiencia.  
Enviar solo lo necesario es una decisión técnica.

## Astro como ejemplo

Astro apuesta por HTML primero y JS bajo demanda.

---

## Conclusión

Optimizar no es exagerar.  
Es respetar el tiempo del usuario.
